# phputils/dev-toolkit

A collection of developer productivity scripts for PHP projects.

## Installation

```bash
git clone https://github.com/YOUR_USERNAME/php-dev-toolkit.git
cd php-dev-toolkit
```

## Available commands

```bash
composer run setup    # Initialise project structure
composer run lint     # Run PHP_CodeSniffer
composer run check    # Validate composer.json + platform checks
composer run analyze  # Run PHPStan static analysis
```

To list all available commands:

```bash
composer list
```
